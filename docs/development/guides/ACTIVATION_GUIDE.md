## 🔧 **TROUBLESHOOTING**

### **"Command not found"**
```bash
chmod +x activate-*.sh setup-api-keys.sh
```

### **"MCP settings not found"**
```bash
ls -la ~/.gemini/settings.json
# If missing, run:
./setup-api-keys.sh
```

### **"GitHub token not working"**
Check: `~/.gemini/settings.json`
Should have: `"GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_..."`
