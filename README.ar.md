## 🚀 البدء السريع

### المتطلبات الأساسية
- Node.js 18+ 
- npm أو yarn
- حساب Supabase
- مفتاح OpenAI API (أو مزود ذكاء اصطناعي بديل)

### 1. استنساخ المستودع
```bash
git clone https://github.com/Moeabdelaziz007/Amrikyy-Agent.git
cd Amrikyy-Agent
```

### 2. إعداد الخادم الخلفي
```bash
cd backend
npm install

# إنشاء ملف .env
cp .env.example .env
# تحرير .env بمعلوماتك (راجع ENV_KEYS_MASTER.md)

# تشغيل خادم التطوير
npm run dev
```

### 3. إعداد الواجهة الأمامية
```bash
cd frontend
npm install

# إنشاء ملف .env
echo "VITE_API_URL=http://localhost:3001" > .env

# تشغيل خادم التطوير
npm run dev
```

### 4. الوصول للتطبيق
- الواجهة الأمامية: http://localhost:5173
- API الخلفي: http://localhost:3001
- فحص الصحة: http://localhost:3001/api/health

## 📦 النشر

### الخادم الخلفي (Railway/Render)

1. **إنشاء مشروع جديد**
   - ربط مستودع GitHub
   - اختيار مجلد `backend` كجذر

2. **متغيرات البيئة**
   - راجع `ENV_KEYS_MASTER.md` للقائمة الكاملة
   - الحد الأدنى المطلوب: 9 متغيرات (راجع الإعداد الأساسي)

3. **النشر**
   ```bash
   # Railway ينشر تلقائياً عند الدفع
   # أو استخدم Railway CLI
   railway up
   ```

### الواجهة الأمامية (Vercel)

1. **استيراد المشروع**
   - ربط مستودع GitHub
   - الإطار: Vite
   - المجلد الجذر: `frontend`

2. **متغيرات البيئة**
   ```bash
   VITE_API_URL=https://your-backend.railway.app
   ```

3. **النشر**
   ```bash
   # Vercel ينشر تلقائياً عند الدفع
   # أو استخدم Vercel CLI
   vercel --prod
   ```

## 🛠️ التقنيات المستخدمة

### الواجهة الأمامية
- **الإطار**: React 18 + TypeScript
- **أداة البناء**: Vite
- **التنسيق**: TailwindCSS + shadcn/ui
- **إدارة الحالة**: React Context + Hooks
- **التوجيه**: React Router v6
- **عميل HTTP**: Axios

### الخادم الخلفي
- **بيئة التشغيل**: Node.js 18+
- **الإطار**: Express.js
- **اللغة**: JavaScript/TypeScript
- **قاعدة البيانات**: Supabase (PostgreSQL)
- **الذاكرة المؤقتة**: Redis
- **المصادقة**: JWT + Passport.js

### الذكاء الاصطناعي والتعلم الآلي
- **الأساسي**: OpenAI GPT-4
- **البدائل**: Gemini، Claude، OpenRouter
- **التتبع**: LangSmith
- **قاعدة البيانات المتجهة**: Qdrant (اختياري)

### DevOps
- **الاستضافة**: Railway (خلفي)، Vercel (أمامي)
- **CI/CD**: GitHub Actions
- **المراقبة**: Sentry
- **التحليلات**: برمجية تحليلات مخصصة
