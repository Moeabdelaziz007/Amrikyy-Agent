## 🆘 COMMON QUESTIONS

### **Q: Where do I start?**
**A:** Read [`README_SIMPLE.md`](README_SIMPLE.md) - 5 minutes, everything you need.

### **Q: How do I get API keys?**
**A:** 
- Gemini: https://makersuite.google.com/app/apikey
- Google Maps: Follow [`GOOGLE_MAPS_SETUP.md`](GOOGLE_MAPS_SETUP.md)
- Stripe: https://dashboard.stripe.com/test/apikeys

### **Q: What's the minimum to run it?**
**A:** Just 3 keys:
```bash
GEMINI_API_KEY=your-key
SUPABASE_URL=your-url
SUPABASE_ANON_KEY=your-key
```

### **Q: How do I test it?**
**A:**
```bash
# Backend
node backend/test-google-maps.js

# Telegram
node backend/telegram-saaaas-bot.js

# APIs
curl http://localhost:3000/api/mini-apps/list
```

### **Q: Where's the code?**
**A:**
```
Backend: backend/src/agents/
Frontend: frontend/src/mini-apps/
Routes: backend/routes/
Services: backend/src/services/
```
